# 🚀 Projeto DevClub - HTML e CSS

Este projeto foi desenvolvido durante a **mentoria do Rodolfo Mori no DevClub** e representa um marco importante na minha jornada de aprendizado em programação.  

Foi aqui que comecei a transformar teoria em prática, aplicando **HTML e CSS** para criar uma página web estruturada, estilizada e responsiva.  

## 🎯 Objetivo
Construir uma página web simples, mas funcional, aplicando:
- Estrutura semântica com HTML
- Estilização com CSS
- Conceitos de responsividade

## 🛠️ Tecnologias utilizadas
- **HTML5**
- **CSS3**
